# Swap Character Typo Extension

A VS Code extension that swaps the last two characters before the cursor.

## Features

- Swaps the last two characters before the cursor in any text file.

## Usage

1. Place your cursor after the two characters you want to swap.
2. Run the command **Swap Last Two Characters Before Cursor** from the Command Palette (Ctrl+Shift+P).
